GuardFile – Secure File Storage System

Overview
GuardFile is a secure file storage application designed to protect user data while maintaining accessibility
and ease of use. The system emphasizes the core cybersecurity principles of confidentiality, integrity, and
availability (CIA triad). Users can securely upload, store, and manage files while receiving feedback on their
overall security posture through an interactive security score.

Requirements - Before running GuardFile, ensure you have the following installed:
Node.js (v16 or higher) (or your actual version)
npm (Node Package Manager)
Git
A modern web browser (Chrome, Edge, Firefox)

Installation
1. Clone the repository: git clone https://github.com/Dylansm37/cpsc491.git

2. Navigate into the project directory: cd cpsc491

3. Install dependencies: 
    npm install
    npm install express

4. Start the server on backend: npm start

5. Start the frontend: npm run dev

6. Open browser of choice and go to http://localhost:5173
